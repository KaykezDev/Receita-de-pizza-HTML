# 🍕 Receita de Pizza - HTML

Página web simples com uma receita de pizza de pepperoni, criada com HTML puro.

## 📋 Sobre o projeto

Exibe uma receita completa de pizza com ingredientes, modo de preparo e um formulário de cadastro de receita.

## 🛠️ Tecnologias

- HTML5
- CSS3 (inline)

## 📄 Funcionalidades

- Imagem da pizza
- Lista de ingredientes
- Passo a passo do preparo
- Formulário de cadastro (receita, autor e e-mail)

## 🚀 Como usar

Basta abrir o arquivo `index.html` no navegador.
